# Capgemini Data Engineering Training Repository

This repository captures my learning journey during the Capgemini Data Engineering Training program. It includes hands-on practice, assignments, and mini projects built using SQL and PySpark, showcasing my progress from basic concepts to building complete data pipelines.

---

## 🔹 Repository Overview

This repository is structured to reflect a step-by-step learning approach, starting from fundamentals and progressing towards real-world data engineering workflows and business analytics.

---

## 🔹 Structure

- **Week-0/**: Core learning phase
  - **Phase-0/**: Fundamentals (Lakeflow, Spark basics, DevOps, Data Ingestion)
  - **Phase-1/**: SQL to PySpark basics and DataFrame operations
  - **Phase-2/**: Data transformation and customer sales analysis
  - **Phase-3/**: Data cleaning and ETL pipeline development
  - **Phase-4/**: Business pipeline and analytics using PySpark
  - **Phase-4A/**: Advanced segmentation techniques (Bucketizer, Quantiles, Window functions)

- **mini-project/**: Reserved for future capstone projects  
- **Week-1 / Week-2 / Week-3/**: Planned for upcoming modules  

---

## 🔹 Key Learnings

- Understanding data engineering workflows (ETL pipelines)
- Working with PySpark DataFrames and SQL queries
- Performing data cleaning, transformations, and aggregations
- Converting SQL logic into PySpark code
- Building end-to-end pipelines for business insights
- Applying segmentation techniques using:
  - Conditional logic  
  - Quantile-based methods  
  - Bucketizer (MLlib)  
  - Window functions
