# PySpark – Data Cleaning & Regex

## 🔹 Objective

In this phase, I worked on cleaning and transforming data using PySpark and applied Regular Expressions (Regex) to extract patterns from text data.

---

## 🔹 Problem Overview

The dataset had:
- Mixed text patterns (alphabets + numbers)  
- Columns like email and phone needing pattern extraction  
- Missing (NULL) values  
- Inconsistent data formats  

Tasks included:
- Cleaning and standardizing data  
- Extracting patterns using Regex  
- Handling NULL values  
- Performing transformations using PySpark  

---

## 🔹 Approach

- Loaded data into DataFrame  
- Identified missing and inconsistent data  
- Applied filtering and transformations  
- Used Regex to extract patterns  
- Cleaned and standardized columns  
- Removed duplicates and sorted data  

---

## 🔹 Key Concepts Used

- Regular Expressions for pattern extraction  
- Data filtering using conditions  
- Column transformations  
- NULL handling  
- Conditional logic  
- Sorting and deduplication  
- String and date functions  

---

## 🔹 Output / Insights

- Clean and structured dataset  
- Extracted email parts and phone details  
- Removed NULL-related issues  
- Standardized text data  
- Prepared data for analysis  

---

## 🔹 Challenges Faced

- Understanding and applying Regex patterns  
- Extracting correct parts from text data  
- Handling mixed data formats  
- Managing NULL values during transformations  

---

## 🔹 Learnings

- Regex helps extract patterns from unstructured data  
- Data cleaning is important before analysis  
- PySpark transformations are useful for real-world data  
- Proper handling of NULL values is necessary  
- Conditional logic helps standardize data  

---

## 🔹 Conclusion

This phase helped me understand how to clean messy data and extract meaningful patterns using PySpark and Regex.
