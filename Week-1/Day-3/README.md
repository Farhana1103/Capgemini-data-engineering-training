# SQL – Conditional Logic & Window Functions

## 🔹 Objective

In this phase, I learned how to apply conditional logic and window functions in SQL to perform advanced data analysis.

---

## 🔹 Problem Overview

Worked with datasets like:
- Employee  
- Sales  
- Customer usage  

Tasks included:
- Applying business logic using `CASE WHEN`  
- Solving nested conditions  
- Performing ranking and partition-based analysis  

---

## 🔹 Approach

- Analyzed datasets and identified required logic  
- Used `CASE WHEN` for conditional transformations  
- Applied nested conditions for complex rules  
- Implemented window functions for ranking  
- Used `PARTITION BY` and `ORDER BY` for grouped analysis  
- Verified outputs using sample data  

---

## 🔹 Key Concepts Used

- `CASE WHEN` for conditional logic  
- Nested conditions for complex rules  
- `ROW_NUMBER()`, `RANK()`, `DENSE_RANK()` for ranking  
- `PARTITION BY` for grouping data  
- `ORDER BY` for sorting within groups  

---

## 🔹 Tasks Performed

- Employee salary categorization  
- Department-wise ranking  
- Customer segmentation based on usage  
- Bonus and salary calculations  
- Identification of high-value and low-value users  

---

## 🔹 Output / Insights

- Ranked employees based on salary  
- Categorized data using business rules  
- Generated grouped and partition-based insights  

---

## 🔹 Challenges Faced

- Writing correct and non-overlapping `CASE WHEN` conditions  
- Deciding proper ranges for categorization  
- Handling nested conditions for multiple rules  
- Understanding difference between `RANK()` and `DENSE_RANK()`  
- Applying correct `PARTITION BY` logic  
- Ensuring correct order of conditions  

---

## 🔹 Learnings

- Strong understanding of conditional logic in SQL  
- Ability to apply real-world business rules  
- Clear understanding of window functions  
- Difference between grouping and partitioning  
- Improved SQL problem-solving skills  

---

## 🔹 Conclusion

This phase helped me understand how to use SQL for advanced analysis by applying conditional logic and window functions to generate meaningful insights.
