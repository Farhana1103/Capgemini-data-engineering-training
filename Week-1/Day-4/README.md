# SQL – Data Cleaning & Functions

## 🔹 Objective

In this phase, I learned how to use SQL functions to clean and transform data, focusing on handling NULL values, formatting text, and working with date data.

---

## 🔹 Problem Overview

The dataset had:
- Missing (NULL) values  
- Inconsistent text formats  
- Date columns requiring analysis  

Tasks included:
- Handling NULL values  
- Cleaning and formatting text data  
- Extracting insights from date columns  
- Applying conditional logic  

---

## 🔹 Approach

- Identified columns with missing and inconsistent data  
- Handled NULL values to avoid incorrect results  
- Cleaned and standardized text data  
- Applied date functions for analysis  
- Used conditions to categorize data  

---

## 🔹 Key Concepts Used

- NULL handling functions  
- String functions for formatting  
- Date functions for extraction and calculation  
- Conditional logic for classification  

---

## 🔹 Output / Insights

- Clean dataset with handled NULL values  
- Standardized and consistent text data  
- Accurate date-based calculations  
- Categorized data based on business rules  

---

## 🔹 Challenges Faced

- Handling NULL values without affecting calculations  
- Choosing correct string functions for formatting  
- Understanding date calculations and differences  
- Applying correct conditions for classification  

---

## 🔹 Learnings

- Data cleaning is important before analysis  
- NULL values can impact results if not handled properly  
- String functions improve data quality and readability  
- Date functions help in time-based analysis  
- Conditional logic helps in categorizing data  

---

## 🔹 Conclusion

This phase helped me understand how to clean and transform data using SQL functions and prepare it for further analysis.
